<h1>What is Array ?</h1>

1 - an array is a collection of items of similar type stored in contiguous memory location.
2 - Sometimes, a simple variable is not enough to hold all data.
3 - example ->
int                   1000, 1004, 1008, 1012, 1016   
 array                [1,  2,  3,  4,  5,  6];
 index                 0   1   2   3   4   5
